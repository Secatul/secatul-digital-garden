## 📚 Comandos por Categoria

### Manipulação de Texto
- [[echo]] - Exibir texto
- [[grep]] - Buscar padrões em texto
- cat - Exibir conteúdo de arquivos
- awk - Processamento de texto
- sed - Editor de stream

### Busca e Navegação
- [[find]] - Buscar arquivos
- locate - Busca indexada
- ls - Listar arquivos
- cd - Navegar diretórios

### Permissões e Segurança
- [[chmod]] - Alterar permissões
- chown - Alterar proprietário
- sudo - Executar como root

### Rede
- ping - Testar conectividade
- netstat/ss - Conexões de rede
- curl - Transferência de dados
- wget - Download de arquivos
